1. `eh_servidor`: É servidor
1. `eh_aluno`: É aluno
1. `eh_prestador`: É prestador
1. `eh_usuarioexterno`: É usuário externo

1. `nome_completo`: Nome completo

1. `data_de_nascimento`: Data de nascimento
